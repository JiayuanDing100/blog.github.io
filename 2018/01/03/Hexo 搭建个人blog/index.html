Hexo 搭建个人blog

第一步：
环境配置要求：

  1. Node
  2. Git
  3. Github

第二步：
安装Hexo： $ sudo npm install -g hexo
在Blog目录下初始化： hexo init
生成静态页面： hexo generate
本地启动服务器： hexo server
那么就可以看到一个简单的页面了

第三部：
建立repository：仓库名必须为【your_user_name.github.io】
修改在Blog下面的_config.yml：
deploy:
type: git
repository: https://github.com/JiayuanDing100/blog.github.io
branch: master

执行如下命令才能用git部署：npm install hexo-deployer-git –save
部署到github：hexo deploy

每次部署：
hexo clean
hexo generate
hexo deploy

一些常用命令：
hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #将.deploy目录部署到GitHub
hexo help #查看帮助
hexo version #查看Hexo的版本

第四步：总结
以后每次写文章
先： hexo new “文章名字”
然后： 在／Blog／sources／_posts／ 编辑文章
再：hexo server

GitHub、Bitbucket、Google Code 优缺点

Bitbucket：与Github相比，最大的特点是 免费支持私有仓库
如果有私人项目或者敏感项目 就选Bitbucket

Github：大部分开源项目都是用的Github，少部分在Bitbucket

Google Code：同时支持 svn / mercurial / git 三种版本管理

Bitbucket, Github, Google Code，分布式版本管理
